lua-resty-ssdb
--------------

The fastest lua client for [ssdb](https://github.com/ideawu/ssdb) and nginx
(based on [lua-nginx-module](https://github.com/openresty/lua-nginx-module)).

Build
------

```bash
$ make
```

Exampple
--------

```lua
local ssdb = require 'ssdb'
```
